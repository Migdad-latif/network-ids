# 🛡️ Network Intrusion Detection System (NIDS)

A research-quality Network Intrusion Detection System built with Python, Scapy, and Machine Learning.

## Project Status
 Under active development

## Overview
This system captures live network traffic, extracts features from packets, detects known attack signatures, flags anomalies using machine learning, and visualises traffic patterns in real time.

## Technology Stack
| Component | Technology |
|---|---|
| Packet Capture | Scapy |
| Anomaly Detection | Scikit-learn |
| Visualisation | Matplotlib |
| Language | Python 3.11 |
